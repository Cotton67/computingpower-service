# 系统六：算力性能监测与优化系统（运维辅助软件）

## 项目说明

本项目为算力服务软件平台“系统六：算力性能监测与优化系统”的前端原型页面，使用 HTML + CSS + JavaScript 开发，不依赖后端服务，使用模拟数据实现业务展示与交互闭环。

## 风格说明

本版本已调整为与样例视频一致的浅色后台管理系统风格：

- 左侧深蓝色固定菜单栏
- 顶部白色标题栏、搜索框、消息、登录按钮
- 主体浅灰背景 + 白色卡片
- 蓝色高亮按钮和菜单
- 每个功能菜单切换为一个独立完整页面，而不是所有模块堆在同一页

## 功能菜单

1. 性能监测驾驶舱
2. 实时性能监测
3. 性能瓶颈诊断
4. 算力优化建议
5. 任务性能分析
6. 历史性能统计
7. 性能告警管理
8. 硬件性能校准
9. 优化效果跟踪
10. 行业性能对标

## 运行方式

直接双击打开 `index.html` 即可运行。

注意：图表使用 Chart.js CDN，首次运行图表需要联网。
