# 系统六：算力性能监测与优化系统（运维辅助软件）

## 项目说明
本项目为算力性能监测与优化系统前端原型，使用 HTML + CSS + JS 开发，可在浏览器中直接运行。使用模拟数据展示算力集群性能、任务性能分析及优化建议。

## 功能模块
- 实时性能监测：GPU利用率、CPU利用率曲线
- 性能瓶颈诊断：显示潜在性能瓶颈
- 算力优化建议：展示优化操作建议，并支持模拟一键优化
- 任务性能分析：显示模拟任务性能数据
- 性能告警管理：显示性能告警信息

## 运行方式
1. 下载或克隆项目文件
2. 打开 `index.html` 文件即可在浏览器中查看系统原型

## 文件结构
```
sys6-kdh/
├── index.html    # 主页面
├── style.css     # 页面样式
├── script.js     # 前端交互与模拟数据逻辑
└── README.md     # 项目说明文档
```

## 使用说明
- 页面加载后，每2秒更新一次模拟性能数据
- 点击“模拟一键优化”按钮，会弹出提示模拟优化操作
- 任务性能表格和性能告警会随模拟数据动态更新
