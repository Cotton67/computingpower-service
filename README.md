# 系统六：算力性能监测与优化系统（运维辅助软件）

## 项目说明

本项目为算力服务软件平台“系统六：算力性能监测与优化系统”的前端原型页面，使用 HTML + CSS + JavaScript 开发，不依赖后端服务，使用模拟数据实现业务展示与交互闭环。

## 本次优化

本版本在浅色后台管理风格基础上增加了各页面内容，避免页面过空：

- 性能瓶颈诊断页新增：瓶颈趋势图、诊断摘要、处理优先级、诊断记录表
- 实时性能监测页新增：节点状态明细表
- 优化建议页新增：优化任务队列表
- 任务分析页新增：任务汇总指标卡片
- 历史统计页新增：历史查询记录
- 告警管理页新增：告警统计卡片
- 硬件校准页新增：校准计划表
- 优化效果跟踪页新增：优化台账
- 行业对标页新增：提升方案

## 风格说明

- 左侧深蓝色固定菜单栏
- 顶部白色标题栏、搜索框、消息、登录按钮
- 主体浅灰背景 + 白色卡片
- 蓝色高亮按钮和菜单
- 每个功能菜单为独立页面切换
