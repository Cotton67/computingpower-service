# 系统六：算力性能监测与优化系统（运维辅助软件）

## 项目说明

本项目为“算力服务软件平台”系统六前端原型，使用 HTML + CSS + JavaScript 开发，不依赖后端服务，使用模拟数据完成页面展示与交互逻辑。

负责人：孔德华

## 功能模块

- 性能总览：GPU、CPU、内存、网络延迟、显存带宽、未处理告警
- 算力性能实时监测：性能趋势图、资源利用分布图、刷新模拟数据
- 性能瓶颈诊断：网络通信瓶颈、显存占用偏高、CPU调度滞后
- 算力优化建议：优化策略列表、模拟一键优化
- 任务性能分析：任务编号、客户、任务类型、GPU卡数、迭代速度、通信耗时
- 历史性能数据统计：周维度性能统计与摘要
- 性能告警管理：告警时间、节点、类型、阈值、级别、处理状态
- 硬件性能校准：标称性能与实际性能对比
- 行业性能对标与优化效果跟踪：行业参考值与优化前后指标对比

## 运行方式

直接双击打开 `index.html` 即可运行。

注意：图表使用 Chart.js CDN，因此首次运行图表需要联网。
