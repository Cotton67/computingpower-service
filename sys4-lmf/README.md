# 算力运营管理平台 - 财务计费结算管理系统

## 项目概述

这是一个基于原生HTML/CSS/JavaScript开发的算力运营管理平台前端原型系统，实现了算力租赁全流程的计费、结算、对账功能，规范财务流程，精准统计营收。

## 核心功能

### 1. 财务运营驾驶舱
- 集中展示算力租赁收入、账单状态、对账进度
- 营收趋势分析与利润分析
- 账单风险监控与财务待办事项
- 实时业务流程可视化

### 2. 计费规则管理
- 支持多种计费模式：按算力性能、按使用时长、按任务量、长期租赁套餐
- VIP客户折扣和长期租赁折扣设置
- 规则模板快速创建
- 规则状态管理（启用/试运行/停用）

### 3. 自动计费模块
- 实时计费计算器，输入用量自动计算费用
- 费用明细拆分展示
- 实时费用明细流水记录
- 智能匹配最优计费规则

### 4. 账单管理
- 自动生成月度、季度、年度账单
- 账单修改、作废、重新生成功能
- 支持Excel/PDF格式导出
- 账单状态跟踪（未支付/已支付/逾期未支付/待确认）

### 5. 结算管理
- 支付登记与对账功能
- 支持多种支付方式（银行转账、企业网银、微信等）
- 财务对账与客户对账双重确认
- 逾期提醒发送功能

### 6. 发票管理
- 电子发票和纸质发票开具
- 税务系统实时同步
- 发票查询、导出、作废功能
- 发票台账管理

### 7. 营收统计分析
- 按客户、按产品、按时间段统计营收
- 年度累计营收与毛利润分析
- 客户营收排行与产品占比分析
- 利润分析报表生成

### 8. 退款管理
- 退款申请审核流程
- 多种退款方式支持（原路退回、银行转账、微信退款等）
- 退款报表生成
- 退款金额记录与状态跟踪

### 9. 财务权限管理
- 分级权限设置（管理员、核算员、出纳、审计查看员）
- 敏感操作审计日志
- 异常登录拦截与监控
- 安全日志导出

## 使用说明

### 快速开始
1. 直接用浏览器打开 `index.html` 文件
2. 系统会自动加载数据并进入财务运营驾驶舱
3. 通过左侧导航栏切换不同功能模块

### 主要操作流程

#### 账单生成流程
1. 进入"计费规则管理"设置计费规则
2. 在"自动计费模块"中使用实时计费计算器
3. 进入"账单管理"点击"自动生成账单"
4. 系统根据用量和规则生成账单
5. 可对账单进行修改、重生成或作废

#### 结算对账流程
1. 客户完成支付后，进入"结算管理"
2. 出纳登记支付信息（金额、方式、时间）
3. 点击"确认到账"完成财务对账
4. 系统自动向客户发送对账确认请求
5. 客户确认后完成对账流程

#### 发票开具流程
1. 账单支付完成后，进入"发票管理"
2. 填写开票信息（客户、类型、金额等）
3. 点击"开具发票"生成电子发票或纸质发票
4. 系统自动同步税务系统
5. 客户可通过邮件或系统下载发票

#### 退款处理流程
1. 客户提交退款申请，进入"退款管理"
2. 财务人员审核退款申请
3. 根据审核结果选择"审核通过"或"驳回申请"
4. 审核通过后推送到财务系统执行退款
5. 记录退款明细并生成退款报表

## 模拟数据说明

系统内置了丰富的模拟数据，包括：

- **5条计费规则** - 覆盖不同计费模式和产品类型
- **5笔客户账单** - 包含不同客户类型和状态的账单
- **5条结算记录** - 记录支付情况和对账状态
- **4张发票记录** - 不同类型和状态的发票
- **3笔退款申请** - 不同原因和处理状态的退款
- **5个财务角色** - 不同权限等级的角色设置
- **5条审计日志** - 记录关键操作历史

这些模拟数据可以完整演示系统的各项功能，也可以作为开发参考数据。
